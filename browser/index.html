Web Deployment

Transform your webpage

This page uses Browserify. Right now, all modules are patched into the bundle, because browserify does not handle dynamic requires, which transformer uses to load its modules.

Once modules are separated into their own proper npm modules (and the modules page works), there will be a module/library to facilitate loading transformer modules directly in the browser with browserify-cdn (like what RequireBin does).

Watch http://github.com/jbenet/transformer to stay tuned! :)
